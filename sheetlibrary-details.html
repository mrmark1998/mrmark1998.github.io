Sheet Music Library

Sheet music library JavaFX app that allows the user to create, read, update, and delete entries on a MySQL server as well as upload sheet music directly to the server. I created this app to easily organize all of the PDF sheet music that I had cluttered up on my computer. It is currently in the works, but the idea is to make it convenient to access my sheet music. It was created using:

  • Intellij IDEA
  • JavaFX
  • SceneBuilder
  • XAMPP server with Apache/MySQL
I integrated Apache common net into the library and was able to use the FTP client protocol to do file upload and download between the client and server.
